1)Configuration
	
	Il faut pouvoir lire les fichiers .js : il faut donc installer nodejs
	Vous pouvez lancer lancer le fichier "node-v4.4.5-x64.msi" pour l'installer
	
	Pour pouvoir utiliser le bot il faut avoir également installé les modules :
		a)express
		b)underscore
		c)weatherapi 
	
	Pour les installer il faut utiliser la commande "npm install <nom>"
	Par exemple "npm install express"
	
2)fonctionnement

	L'application sé déclenche en envoyant un tweet avec une localisation(option du tweet) avec le texte @aze141414 et en lancant "node ProjetL010.js"
	Le bot va répondre au dernier tweet envoyé et renvoyer un tweet à l'utilisateur avec 4 informations dont la ville choisie, la température en degrés celsius,
	le pourcentage d'humidité et les milimètres de précipitation si une localisation a été envoyé avec le tweet. Sinon il renverrra un message d'erreur
